How weights distribute
// for each assessment A weight_per_LO = A.weight ÷ count(A.LOs) // for each LO total_LO_weight = Σ weight_per_LO for every A tagged with LO // pass criterion student earned ≥50% of total_LO_weight

Each assessment's weight is split evenly across the LOs it covers. A student passes an LO when they earn at least 50% of that LO's total available weight across all assessments.
